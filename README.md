#### 概要
キャンバスに描いた絵をネコ、サメ、ペンギン、タコの4種類から識別するWebアプリ

#### 作業内容
* データセットからネコ、サメ、ペンギン、タコの4種類の画像をそれぞれから7万枚ずつ読み込みCNN(畳み込みニューラルネットワーク)を用いて学習させた画像分類モデルを作成
* 学習したモデルを使用しGradioでアプリを作成、アプリ上で絵を描き識別させる
